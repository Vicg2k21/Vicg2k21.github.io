1/6 Scale Collectables Store

Live Demo

Description

This is a desktop application developed using C# and Visual Studio 2022. The system supports multiple user roles including managers, employees, and customers. Users can browse products, manage orders, and complete purchases, while managers can control inventory, accounts, and generate reports. The application demonstrates object-oriented programming, database integration, and user interface design.

Key Features

  • User authentication (manager, employee, customer, or guest)
  • Browse collectible products with images
  • Add items to cart and checkout system
  • Apply promo codes for discounts
  • View order history
  • Manager controls for inventory, accounts, and reports
  • POS (Point of Sale) checkout functionality

Technologies Used

  • C#
  • .NET Framework
  • Visual Studio 2022
