Description

This is a desktop application developed using C# and Visual Studio 2022. The application allows users to browse collectible items, interact with the interface, and simulate an online shopping experience. It demonstrates object-oriented programming, user interface design, and data handling.

Key Features

  • Browse collectible products
  • User interaction through GUI
  • Data handling and display
  • Organized application structure

Technologies Used

  • C#
  • .NET Framework
  • Visual Studio 2022
