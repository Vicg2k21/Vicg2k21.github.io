Travel Agency Lodging System

Demo

Description

This is a Java-based desktop application developed using Apache NetBeans. The system allows users to browse different lodging options such as houses and hotels, view images, and select start and end dates for their stay. The project demonstrates object-oriented programming, file handling, and GUI design.

Key Features

  • Select lodging type (house or hotel)
  • Choose start and end booking dates
  • Display lodging images
  • Simple and interactive user interface

Technologies Used

  • Java
  • Apache NetBeans IDE
  • Object-Oriented Programming (OOP)
